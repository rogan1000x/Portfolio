# Day 03 - 포트폴리오 디자인 개선 (사이버펑크 테마 강화)

## 오늘 배운 것
- **배경 그라데이션**: linear-gradient로 동적인 배경
- **background-attachment: fixed**: 스크롤해도 배경 고정
- **Box-shadow 강화**: inset 그림자로 깊이감 추가
- **CSS 애니메이션**: @keyframes로 섹션 헤더 글로우 효과
- **포트폴리오 구조 설계**: 메인은 통일감, 각 프로젝트는 독립적 디자인

## 오늘 한 것
### 1. 배경 개선
```css
/* 단색에서 그라데이션으로 */
background: linear-gradient(135deg, #0d0015 0%, #1a0033 50%, #0d0015 100%);
background-attachment: fixed;  /* 스크롤해도 배경 고정 */
```
- 더 생동감 있는 느낌
- 사이버펑크 테마 강화

### 2. 프로젝트 카드 글로우 강화
```css
.project-card:hover {
  background: rgba(255, 45, 155, 0.15);
  box-shadow: 0 0 40px rgba(255, 45, 155, 0.8),      /* 핑크 글로우 강화 */
              0 0 40px rgba(0, 255, 255, 0.6),        /* 시안 글로우 추가 */
              inset 0 0 20px rgba(255, 45, 155, 0.1); /* 내부 그림자 */
  transform: translateY(-8px);  /* -5px → -8px */
}
```
- 호버 시 더 강렬한 네온 효과
- inset 그림자로 3D 깊이감

### 3. 섹션 헤더 애니메이션
```css
@keyframes sectionGlow {
  0%, 100% {
    text-shadow: 0 0 15px #ff2d9b;
  }
  50% {
    text-shadow: 0 0 25px #ff2d9b, 0 0 35px rgba(255, 45, 155, 0.5);
  }
}

section h3 {
  animation: sectionGlow 3s ease-in-out infinite;
}
```
- "About", "Projects", "Skills" 제목이 살짝 깜빡거림
- 시선을 자연스럽게 끌음

## 포트폴리오 구조 설계 (향후 확장성)

### 현재 상태
포트폴리오 메인 (사이버펑크)
└─ 게임 카드 → https://cyberfunk-hangle.netlify.app/

### 향후 프로젝트 추가 예시
나중에 새로운 프로젝트를 만들 때:
포트폴리오 메인 (사이버펑크 - 유지)
├─ 게임 카드 → (사이버펑크 디자인)
├─ 날씨앱 카드 → (독립적 디자인)
└─ 투두리스트 카드 → (독립적 디자인)

**원칙**: 포트폴리오 메인은 통일감 유지, 각 프로젝트는 독립적 디자인

**핵심:**
- **포트폴리오 메인**: 개발자의 정체성 (통일된 디자인)
- **각 프로젝트**: 프로젝트의 목적에 맞춘 독립적 디자인

## 디자인 철학
1. 포트폴리오 메인 = 관문 역할 (내 스타일 표현)
2. 각 프로젝트 = 독립적 포트폴리오 (프로젝트별 개성)
3. 링크로 연결 (서로 간섭 없음)
