# Portfolio Day 01 - 포트폴리오 사이트 기본 구조 완성

## 오늘 배운 것
- **포트폴리오 구조 설계**: 프로젝트와 독립적으로 관리하기
- **Grid 레이아웃**: CSS Grid로 반응형 카드 배치
- **Intersection Observer**: 스크롤 시 섹션 나타나기 애니메이션
- **CSS 네온 효과**: text-shadow로 사이버펑크 테마 구현
- **버튼 상태**: hover 상태로 인터랙션 추가

## 오늘 한 것
- Portfolio 저장소 생성 (GitHub)
- 폴더 구조 설계:
  - 프로젝트는 외부 링크로 연결
  - 포트폴리오는 독립적으로 관리
  
- HTML 구조:
  - header: 자기소개 (ROGAN KIM, Web Developer)
  - about: 간단한 소개
  - projects: 프로젝트 카드 (게임 포함)
  - skills: 기술 스택 그리드
  - footer: 연락처

- CSS 스타일:
  - 사이버펑크 테마 (핑크#ff2d9b + 시안#00ffff)
  - 프로젝트 카드 hover 효과
  - 반응형 디자인 (모바일 최적화)
  - text-shadow로 네온 글로우

- JavaScript 기능:
  - 페이지 로드 시 헤더 페이드인 애니메이션
  - 스크롤 시 섹션 나타나기 (IntersectionObserver)
  - 프로젝트 카드 클릭 효과음
  - 부드러운 앵커 스크롤
